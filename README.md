## PREECLAMPSIA-PREDICTION
# Project Overview

This project is an AI-powered clinical decision support system that predicts the risk of Preeclampsia using machine learning models and temporal analysis.The system compares a patient’s previous visit report and current visit report to analyze how the risk changes over time. It helps in identifying worsening clinical conditions early.This project was developed as an academic healthcare AI project using Python, Machine Learning, and Streamlit.

# Features

1. Predicts preeclampsia risk using Machine Learning
2. Uses multiple ML models for better accuracy
3. Temporal analysis for tracking patient condition over time
4. Upload previous and current clinical reports
5. Displays risk trend changes
6. Interactive Streamlit dashboard
7. Feature importance visualization
8. Clinical risk alerts

# Machine Learning Models Used
